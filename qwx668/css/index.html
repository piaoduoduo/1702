    海洋污染

    大多数陆地和大气中的污染物最终都以不同方式进入海洋。陆地上的工农业活动及城乡居民的日常生活产生的大量废水、热废水和垃圾，通常通过河流、排污口或直接排放进入海洋，构成了海洋污染的直接来源；而陆地工农业活动排放的大量细小颗粒和废气则通过大气传播，最终沉降到海洋，成为海洋污染的间接来源。

    人类活动

    经过几十年的调查和监测结果显示，来自陆地的污染物占中国近海污染总量的80%以上。因此，减少陆源污染物流入海洋是防治海洋污染的关键。海洋作为地球表面最低的区域，几乎所有地表污染物都能进入海洋。因此，海洋污染物种类繁多且复杂，涵盖了有机污染物、石油及其衍生物、重金属、农药、放射性物质等。
